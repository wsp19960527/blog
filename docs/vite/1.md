## 安装

搭建 vite 项目

```bash

//初始化一个vite项目
pnpm create vite

// 使用模板
pnpm create vite@latest my-vue-app -- --template vue

// 手动安装vite
pnpm add -D vite

```

并创建一个像这样的 index.html 文件：

```html
<p>Hello Vite!</p>
```

然后终端运行

```bash
pnpm vite
```
